# Node-Red

Este documento contiene las instrucciones para la operación de Node-Red.

## Requisitos

- Node.JS
- NPM
- Node-Red

Para iniciar el servidor, ejecuta el comando `node-red` en una terminal y mantén la terminal abierta durante el periodo de operación. Para correr node-red, asegurate de tener el directorio de instalación agregado al path del sistema.
